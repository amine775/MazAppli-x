BIENVENUE SUR MAZAPLI-X

Auteur : Amine Maza 

Mazapli-X c'est quoi? 
Et bien c'est une application android qui te permet de connaitre la météo dans la ville de ton choix. 
Il te suffit d'entrer le nom de ville et le tour est joué !

Date clé :
Le projet était en développement depuis novembre 2019 et a été livré le 23/12/2019

A quoi ressemble l'application :
Et bien pour faire simple, Mazappli possède un simple écran ou on entre la ville et visualise
les données en même temps, plutôt pratique et rapide ! (On veux pas faire compliquer pour vous dire
qu'il pleut, c'est assez dur pour vous...)

Comme mentionné précédement l'applicatione est désormais livré, mais on peut toujours l'améliorer et
y apporter un tas de nouvelles fonctionnalitées !! (On fera quand même au plus simple)


L'application est compatible uniquement sur Android et à été développé en KOTLIN. 
Architecture utilisée : MVC
Base de données : Non
Contenu : Une activité et deux fragment
Design : plutôt epurée avec de belles images qui apparaissent en fonction du temps (c'est beau la technologie)
API utilisé :  openweathermap.com (fourni les données météo)




Application réalisé pour un TP à l'ESIEA !
